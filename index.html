Download and extract the package into your usr/local directory and run the below lines in terminal to be able to start your localhost with the command line.

> echo 'export PATH=/usr/local/apache/bin:$PATH' >> ~/.bash_profile > echo 'export MANPATH=/usr/local/apache/man:$MANPATH' >> ~/.bash_profile > source ~/.bash_profile

Now, to start/stop Apache, all you have to do is:

> sudo apachectl start > sudo apachectl stop
